# create-hana

A modern, flexible project scaffolding tool for JS/TS projects.

## Features

- Any language
- Any framework
- Any linter & formatter
- Any package manager
- Any bundler
- ...****

## Usage

```bash
pnpm create hana
```

Currently supports:
